# Estado del Consentimiento Web en Chile 2026

Estudio empírico open-source de las prácticas de consentimiento de cookies
en sitios chilenos, motivado por la entrada en vigor de la Ley 21.719 en
diciembre de 2026.

Replica metodológica adaptada de Nouwens et al. (CHI '25), con extensiones:
captura de cookies en tres escenarios (`baseline`, `aceptar`, `rechazar`),
clasificación por agentes LLM con caché versionada, y reporte determinístico
sin invocaciones LLM en la fase de análisis.

> Repositorio en formación. Ver el primer Pull Request para la entrega
> inicial de scripts, datos y reporte.
